# Skills

## cx-rag-researcher

Answers questions about Mozilla user experience by querying three data sources via semantic vector search:

| Source | What it contains | Use for |
|--------|-----------------|---------|
| SUMO / Kitsune | Firefox user forum threads | Sentiment, user feedback |
| Zendesk | Support tickets and bug reports | Problem patterns, bugs |
| Knowledge Base | Official Mozilla support articles | Guidance, solutions |

**Example questions:**
- What are users complaining about with Firefox sync?
- What are the top pain points for Firefox on Android in Q1 2026?
- What does official guidance say about password manager issues?

**Invoke:** Just ask Claude a question about Mozilla user experience — the skill triggers automatically.
